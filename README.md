# Scripter Garden Horizons Macro
## Installation:
1. Download [AutoHotKey v1](https://autohotkey.com)
2. Install the macro from the [latest release](https://github.com/DeweyPointJr/Scripter-Garden-Horizons-Macro/releases/latest)
3. Extract the zip file

## Usage:
1. Run Macro.ahk
2. Use the menu in the macro to select which items to buy, and press start
3. Press F3 (or another keybind if you changed it) to stop the macro.

## Features:
 - Seed Shop
 - Gear Shop
 - Auto Harvest
 - Auto Sell
 - Botanist Donations
 - Auto Reconnect
 - Auto Update
